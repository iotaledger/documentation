# Qubic

Qubic is a protocol that specifies IOTA's solution for oracle machines, smart contracts, and outsourced computations.  As of February 2019, development is underway.  Please see the development [roadmap](https://qubic.iota.org/roadmap).

## Components

- [Qubic software](0.1\qubic\README.md)

- [**Q**ubic **C**omputation **M**odel (QCM)](https://github.com/iotaledger/qcm)

- **I**ota **C**ontrol agen**T** (ICT) - light-weight node
    
- **I**ota e**X**tension **I**nterface (IXI)
    
- Abra dataflow language
    
- [QUPLA - **QU**bic **P**rogramming **L**anguage](https://github.com/iotaledger/qupla)
