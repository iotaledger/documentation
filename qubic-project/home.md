# Qubic

Qubic is a protocol that specifies IOTA's solution for oracle machines, smart contracts, and outsourced computations.  This documentation is being written while the system is being developed.  For details, see the [Qubic Roadmap](https://qubic.iota.org/roadmap).

## Components

- [Qubic](0.1/qubic/README.md)

- [**Q**ubic **C**omputation **M**odel (QCM)](https://github.com/iotaledger/qcm)

- **I**ota **C**ontrol agen**T** (ICT) - light-weight node
    
- **I**ota e**X**tension **I**nterface (IXI)
    
- [Abra dataflow language](0.1/qubic/concepts/abra.md)
    
- [QUPLA - **QU**bic **P**rogramming **LA**nguage](https://github.com/iotaledger/qupla)
